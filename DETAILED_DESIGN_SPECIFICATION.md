**Элементы Header (слева направо):**

1. **☰ Burger Menu** (левый верхний угол)
   - Иконка "три полоски" (24x24px)
   - При клике открывается боковая панель навигации
   - Цвет: Темно-синий (#2E1A7C)

   - "Академия ПСБ" с иконкой

3. **Поиск** (центр, опционально)
   - Поисковая строка с иконкой 🔍
   - Поиск по курсам, материалам, заданиям

4. **Профиль пользователя** (самый правый край header)
   - Имя "Иван И." слева
   - Круглый аватар (40x40px) справа
   - Dropdown при клике:
     - Личный кабинет
     - Выйти

### Дашборд - Детальное описание секций:

#### 1. **Статистика (Dashboard Cards)**
- **4 карточки в ряд** (на десктопе)
- **Размер**: Равная ширина, высота 140px
- **Фон**: Белый с легкой тенью
- **Содержимое каждой карточки**:
  - Крупная цифра (48px, жирный шрифт)
  - Описание под цифрой
  - Иконка (опционально)

  **Карточки**:
  1. **Активных курсов**: количество курсов, в которых студент зачислен
  2. **Домашних заданий**: выполненных/всего заданий (например, 2/10)
  3. **Средний балл**: средний процент успеваемости
  4. **Дней подряд**: количество дней активности подряд с иконкой огня 🔥 (геймификация)

#### 2. **Продолжить обучение**
- **Формат**: Одна большая горизонтальная карточка
- **Логика**: Показывает последний открытый курс/урок
- **Элементы**:
  - Превью изображение курса (слева, 120x120px)
  - Название курса
  - Текущий модуль/урок
  - Progress bar (цветной, анимированный)
  - Кнопка "Продолжить обучение" (оранжевая, CTA)

#### 3. **Лучшие курсы (Слайдер)**
- **Формат**: Слайдер с изображениями курсов
- **Высота**: 400px
- **Количество слайдов**: 5 лучших курсов
- **Автоматическая прокрутка**: Смена слайда каждые 5 секунд
- **Элементы слайдера**:
  - Фоновое изображение курса (background-image)
  - Gradient overlay (темный градиент снизу для читаемости текста)
  - Badge в левом верхнем углу (например, "Топ курс", "Новинка", "Популярный")
  - Название курса (белый текст, крупный шрифт, внизу)
  - Описание курса (белый текст, меньший шрифт, внизу)
  - Навигация:
    - Точки (dots) снизу - текущий слайд выделен (более длинная точка)
    - Стрелки влево/вправо (круглые кнопки с символами ‹ ›)
  - При клике на слайд → переход на страницу курса

#### 4. **Каталог курсов**
- **Формат**: Фильтруемая и сортируемая сетка курсов
- **Элементы управления**:

  **Поиск**:
  - Поисковая строка (🔍 Поиск курсов...)
  - Поиск по названию и описанию курса

  **Категории** (кнопки-таблетки):
  - Все курсы (по умолчанию активна)
  - Программирование
  - Дизайн
  - Бизнес
  - Маркетинг
  - Data Science
  - Активная категория выделена цветом (фон - цвет primary, текст - белый)

  **Сортировка** (выпадающий список):
  - Популярные (по умолчанию, по количеству студентов)
  - Новые (по дате создания)
  - По рейтингу (от высшего к низшему)
  - По названию (алфавитный порядок)

  **Фильтр по длительности** (выпадающий список):
  - Все
  - Короткие (< 30 часов)
  - Средние (30-60 часов)
  - Длинные (> 60 часов)

  **Фильтр по сложности** (выпадающий список):
  - Все уровни
  - Начальный
  - Средний
  - Продвинутый

- **Сетка курсов**: Grid 3-4 карточки в ряд (адаптивная)
- **Карточка курса**:
  - Обложка курса (16:9, 180px высота)
  - Название курса
  - Имя преподавателя с иконкой 👤
  - Progress bar (если курс активный)
  - Процент прохождения (если курс активный)
  - При наведении: эффект подъема (hover)
  - При клике → переход на страницу курса

## 🚀 КЛЮЧЕВЫЕ ПОЛЬЗОВАТЕЛЬСКИЕ СЦЕНАРИИ

### **Сценарий 1: Студент изучает новый урок**
1. Логин → Дашборд
2. Видит "Продолжить обучение" → клик
3. Открывается страница урока с видео
4. Смотрит видео, делает заметки
5. Отмечает урок как пройденный
6. Автопереход к следующему уроку

### **Сценарий 2: Студент выполняет домашнее задание**
1. Burger Menu → Домашние задания
2. Видит дедлайны, выбирает срочное задание
3. Читает условие задания
4. Загружает файлы (drag & drop)
5. Добавляет комментарий
6. Сохраняет черновик (автосохранение работает)
7. Проверяет еще раз → отправляет на проверку
8. Получает уведомление "Отправлено!"

### **Сценарий 3: Студент получает обратную связь**
1. Уведомление 🔔 "Ваша работа проверена!"
2. Переход к результатам
3. Видит оценку 85/100, разбивку по критериям
4. Читает комментарии преподавателя
5. Видит инлайн-комментарии к коду
6. Задает вопрос преподавателю в чате
7. Решает исправить и отправить новую версию
8. Переход к форме загрузки, отправляет v2

### **Сценарий 4: Студент общается с преподавателем**
1. Burger Menu → Чат
2. Выбирает личный чат с преподавателем
3. Задает вопрос по заданию
4. Прикрепляет скриншот с ошибкой
5. Получает ответ от преподавателя
6. Преподаватель отправляет пример кода
7. Студент благодарит, применяет решение

### **Сценарий 5: Студент отслеживает свой прогресс**
1. Burger Menu → Успеваемость
2. Видит общую статистику (85% средний балл)
3. Смотрит график прогресса (растет!)
4. Разворачивает детали курса Python
5. Видит все оценки по заданиям и тестам
6. Экспортирует отчет в PDF
7. Мотивирован продолжать обучение!
