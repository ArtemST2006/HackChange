# Changelog

## Версия 1.2.0 (2025-11-29)

### Изменения в Header - Профиль пользователя

#### Обновлено:
- ✅ **Расположение**: Профиль теперь прижат к самому правому краю header
- ✅ **Порядок элементов**: Имя пользователя слева, аватар справа (было наоборот)
- ✅ **Dropdown меню упрощено**:
  - Только 2 пункта: "Личный кабинет" и "Выйти"
  - Удалены: "Мой профиль", "Успеваемость", "Настройки"

#### Новая структура dropdown:
```
┌─────────────────────┐
│ Личный кабинет      │
│ Выйти               │
└─────────────────────┘
```

#### CSS изменения:
- `.header-right` - добавлен `margin-left: auto` для прижатия к правому краю
- `.header-profile` - добавлен `padding-right: 0` для точного выравнивания
